# Helix
Multi-layer digital media forensic system for detecting AI-generated and manipulated content.

## Overview
Aegis-Provenance analyzes media using multiple independent techniques such as PRNU, entropy analysis, physiological signals, and probabilistic fusion to verify authenticity.

## Pipeline
Input → Feature Extraction → PRNU → Entropy → Physiological → Fusion → Output

## Key Concepts
Digital Signal Processing (DSP)
PRNU (Camera Fingerprinting)
Entropy Analysis
Physiological Signal Detection
Bayesian / Dempster-Shafer Fusion

## Tech Stack
Python, OpenCV, NumPy, SciPy, Streamlit

## How to Run

pip install -r requirements.txt
python main.py
